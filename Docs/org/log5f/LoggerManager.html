forceCompile()method
public static function forceCompile(someClass:Class):void

This method does nothing, it used for force compile specified class. You can create custom appenders, filters, formatters and layouts, but they are not using in project, usually. Therefore flash compiler doesn't compile they into result swf. For resolving this issue you can use this method. 

   LoggerManager.forceCompile(MyAppender);

someClass:Class — A some class that is not used explicitly in project, but can be instantiated during run time.
