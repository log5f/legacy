Method detail
decide()method
public function decide(event:LogEvent):int

If the decision is DENY, then the event will be dropped. If the decision is NEUTRAL, then the next filter, if any, will be invoked. If the decision is ACCEPT then the event will be logged without consulting with other filters in the chain. Parameters

event:LogEvent — The LoggingEvent to decide upon.

Returns
int — decision The decision of the filter.
